# QGIS 실내 도면 제작 가이드

제1공학관(ENG1) GeoJSON을 QGIS에서 만드는 step-by-step 가이드.

**핵심 원칙: QGIS에서는 지오메트리(도형)만 정확하게 그린다. 속성(방 번호, 타입 등)은 Python 스크립트로 일괄 입력한다.**

---

## 1. QGIS 설치

1. https://qgis.org → **QGIS LTR (Long Term Release)** 다운로드/설치
2. 설치 후 실행

---

## 2. 맵 불러오기

1. 왼쪽 **탐색기** 패널 → **XYZ Tiles** 우클릭 → **OpenStreetMap**을 더블클릭해서 레이어에 추가
2. 제1공학관 위치로 이동: 하단 **"위치를 찾으려면 입력하십시오"** 바에 `14134827.7, 4479925.0` 입력, 축척 1:5000

---

## 3. 이미지 붙이기

구조도 JPG를 위성사진 위에 정확히 겹쳐놓는 과정.

1. 메뉴 -> 플러그인 -> 플러그인 관리 및 설치
2. Freehand raster georeferencer 플러그인 설치
3. 메뉴 -> 래스터 (R) -> Freehand Raster Georeferencer -> Add raster for interactive georeferencing
4. 사진 맞추기
5. 레이어 패널에서 투명도 조절

---

## 4. 사전설정

정확한 벽 연결을 위한 스냅 설정.

1. 보기 -> 패널 -> 고급 디지타이즈 작업
2. 보기 -> 툴바 -> 스냅 툴바
3. 보기 -> 툴바 -> 형테 디지타이징 툴바
3. 스냅 툴바에서 자석 아이콘 클릭, 꼭짓점/선분 체크
4. 고급 디지타이즈 작업에서 삼각자 아이콘 클릭, 톱니바퀴 아이콘으로 각도 스냅되게 설정

---

## 5. 레이어 생성

### 5.1 건물 외곽선 레이어
1. **레이어** → **레이어 생성** → **새 Shapefile 레이어**
2. 설정:
   - File name: `eng1_outline.shp`
   - Geometry type: **Polygon**
   - CRS: **EPSG:4326 (WGS 84)**
3. 속성 필드: **추가 안 해도 됨** (스크립트에서 자동 입력)
4. OK

### 5.2 방(Rooms) 레이어
- File name: `eng1_rooms_L1.shp`
- Geometry type: **Polygon**
- CRS: **EPSG:4326**
- 속성 필드: **없음** (스크립트에서 자동 입력)

### 5.3 복도(Corridors) 레이어
- `eng1_corridors_L1.shp`, Polygon, EPSG:4326
- 속성 필드: **없음**

### 5.4 벽(Walls) 레이어
- `eng1_walls_L1.shp`
- Geometry type: **LineString**
- CRS: **EPSG:4326**
- 속성 필드: **없음**

---

## 6. 외각선 따기

### 6.1 건물 외곽선
1. `eng1_outline` 레이어 선택 → **편집 모드 켜고끄기** (연필 아이콘 ✏️)
2. **폴리곤 피처 추가** 선택
3. 위성사진/구조도에서 건물 외곽을 따라 클릭 (모서리마다)
4. 다 그리면 **우클릭**으로 완료 (잘못 찍었으면 Backspace)
5. 속성 팝업이 뜨면 그냥 OK (비워두기)
6. **레이어 편집 내용 저장** (💾)

### 6.2 방 (3점 직사각형 + 피처 분할)

#### Step 1: 3점 직사각형으로 큰 영역 그리기
1. `eng1_rooms_L1` 레이어 선택 → **편집 모드 켜고끄기** (연필 아이콘)
2. 형태 디지타이징 툴바 → **직사각형 아이콘 옆 ▼** → **3점에서 직사각형 추가**
3. 첫 점: 외곽선(또는 기존 방) 꼭짓점에 스냅
4. 둘째 점: 외곽선을 따라 다른 꼭짓점에 스냅 (한 변 방향 결정)
5. 셋째 점: 높이 방향으로 드래그 → 클릭
6. 속성 팝업 → 그냥 OK

#### 직사각형 모드 해제
- 디지타이징 툴바의 **▼ 드롭다운** → **선분으로 디지타이즈** 선택하면 일반 폴리곤 모드로 복귀

#### Step 2: 피처 분할로 방 나누기
1. **피처 분할** 선택 (단축키 또는 **편집** → **도형 편집** → **피처 분할**)
2. 윗변 선분에 스냅해서 첫 점 클릭
3. **`A`** → **`90`** → 🔒 잠금 — 윗변 기준 직각 유지
4. 아랫변을 관통하도록 클릭 → 우클릭 완료
5. 반복해서 방 개수만큼 분할
6. **레이어 편집 내용 저장** (💾)

#### 계단 영역
- 계단도 `eng1_rooms_L1`에 폴리곤으로 그린다 (별도 레이어 X)
- 속성 입력 시 `room_type: "stairs"`로 구분

### 6.3 복도
- 같은 방법으로 복도 영역을 폴리곤으로 그리기

### 6.4 벽
- `eng1_walls_L1` 레이어 → **라인 피처 추가**
- 벽을 따라 LineString 그리기


---

## 7. GeoJSON 내보내기

### 레이어별 내보내기
1. 레이어 우클릭 → **내보내기** → **다른 이름으로 피처 저장...**
2. 설정:
   - Format: **GeoJSON**
   - CRS: **EPSG:4326**
   - File name: 예) `eng1_rooms_L1.geojson`
3. OK

**모든 레이어를 개별 GeoJSON으로 내보내기:**
- `eng1_outline.geojson`
- `eng1_rooms_L1.geojson`
- `eng1_corridors_L1.geojson`
- `eng1_walls_L1.geojson`
